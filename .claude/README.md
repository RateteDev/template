# .claude

Claude Code 向けのプロジェクト設定を格納する。

## Hooks

`settings.json` の `SessionStart` フックで、セッション開始時に以下を自動出力する。

- `just --list` — 利用可能なコマンド一覧
- `find docs -type f` — `docs/` 配下のファイル一覧（ディレクトリ単位でグルーピングして出力）

いずれもファイル名・コマンド名を一覧表示するだけで、内容までは読み込まない。詳細が必要な場合は該当ファイルを都度参照する。

出力は `=== セクション名 ===` の下に `$ 実行コマンド` を続け、何の実行結果かを明示する。今後セクションを追加する際もこの形式に合わせる。

実体は justfile の `_session-start` レシピ（`[private]`）であり、`settings.json` からはそれを呼び出すのみ。動作を手元で確認したい場合は `just _session-start` を実行すればよい。
