# Contributing

## Development Workflow

1. Create a branch from `main`.
2. Write tests first (TDD).
3. Implement the feature or fix.
4. Run the relevant app's check and test recipes to verify.
5. Commit following the rules in `docs/rules/コミットルール.md`.
